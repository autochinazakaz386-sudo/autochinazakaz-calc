Калькулятор растаможки авто под ключ в Уссурийске

Усиленная автономная HTML-страница для расчёта стоимости автомобиля из Китая и других рынков с учётом валюты покупки, пошлины, НДС, акциза, утильсбора, комиссий ВТБ, экспортёра, брокера, СВХ и прочих затрат.
1 файл Открывается локально Автообновление курсов Сохранение расчётов Печать / экспорт JSON
Встроены два режима расчёта: детальный импорт с раздельными налогами и личный ввоз по единым ставкам для авто с ДВС/гибридов. Все спорные или меняющиеся ставки можно корректировать вручную без правки кода.

1. База сделки

Цена авто, валюта покупки, возраст, двигатель и тип силовой установки.

2. Режим расчёта таможни

Переключай методику расчёта под кейс клиента.
В этом режиме страница отдельно считает таможенную пошлину, НДС, акциз и утильсбор. Для коммерческих кейсов блоки пошлины и утильсбора сделаны редактируемыми, потому что итог может зависеть от точного кода ТН ВЭД, документов и финальной классификации брокером.
Для личного ввоза авто с ДВС и гибридов применяется единая ставка по возрасту и рабочему объёму двигателя. Для электромобилей и водородных авто в этой странице оставлен редактируемый адвалорный пресет, так как расчёт по объёму двигателя к ним не применяется.

3. Комиссии и расходы

Все дополнительные траты до выдачи авто в Уссурийске.

4. Курсы валют

Страница пытается получить живые курсы онлайн. Если API недоступен из браузера, можно работать на текущих значениях и править вручную.

Все суммы пересчитываются в рубли. Для личного ввоза единая ставка показывается отдельно от НДС и акциза.
